Please note that this project was generated using Vite for smaller bundle sizes for our project.
  
   npm create vite@latest

Bootstrap was also used for icons and styling
   npm install bootstrap@latest
   npm install react-bootstrap-icons --save

We are using ReactFlow for the Component Flow Chart.
   npm install reactflow

And Chakra UI for enhanced look and feel.
   npm install @chakra-ui/react @emotion/react @emotion/styled framer-motion
